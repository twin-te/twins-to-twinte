# Twins to Twin:te
twinsからtwinteへ履修授業をインポートするスクリプト

# 動作条件
- モダンブラウザ
- twins履修状況を表示するページでスクリプトをロードすること

# 動作原理
`Twin:teにインポート` ボタンが追加されます。
ボタンを押すと、履修状況ページのDOMから授業番号を取得し、twinteに登録します。
